# CLAUDE.md

## 项目概览

`chat_box_code` 是一个 **AI Agent Hub**，支持多个智能代理协作完成复杂任务，例如：

- 旅行规划与行程生成
- 深度研究与报告生成
- 多步骤推理任务

项目重点是 **模块化 Agent 架构**，每个 Agent 可以独立演进，同时共享核心基础设施，而不仅仅是一个聊天界面。

---

## 范围与职责

仓库提供：

- **前端**：Next.js + React + TypeScript + Tailwind CSS
- **Agent 页面**：每个 Agent 对应一个页面，位于 `src/app/**`
- **API 路由**：Agent 后端逻辑在 `src/app/api/**`
- **工具与辅助函数**：用于 Prompt 处理、流式输出和数据获取
- **文档**：可选的 Agent 说明文档位于 `docs/`

Agents 彼此独立，但可共享工具函数。修改或新增功能时应遵循 Agent 边界原则。

---

## 架构说明

- **LLM 与外部服务**：通过环境变量配置（参考 `.env.example`）
- **流式聊天与交互处理**：通过 API 路由集中管理，模块化设计
- **部署方式**：支持 Vercel、Docker 或本地开发（`npm run dev`）
- **设计原则**：
  - Agent 之间边界清晰
  - 尽量减少跨 Agent 依赖
  - 优先采用增量改进而非大幅重构

---

## 使用与导航

不需要一次性理解所有 Agents，使用时遵循 **渐进式披露**：

- 查看某个 Agent 功能 → 查阅对应页面和 API 路由
- 理解环境变量配置 → 查看 `.env.example`
- 查看整体运行或部署方式 → 查阅 `README.md` 和 `docs/`

优先关注当前任务相关部分，再逐步深入其他模块。

---

## 设计原则

1. 清晰优先于聪明
2. 保持 Agent 独立性
3. 注重可维护性和增量开发
4. 除非必要，避免过度抽象
5. 新增 Agent 或重大修改需在 `docs/` 中记录
